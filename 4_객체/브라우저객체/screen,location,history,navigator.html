Screen

사용자의 모니터 정보를 제공하는 객체

  1. screen.width : 화면의 너빗값
  2. screen.height : 화면의 높이값
  3. screen.availWidth : 작업표시줄을 제외한 화면의 너빗값
  4. screen.availHeight : 작업표시줄을 제외한 화면의 높잇값
  5. screen.colorDepth : 사용자 모니터가 표현 가능한 컬러 bit

location

사용자 브라우저와 관련된 속성과 메서드 제공 (현재URL에 대한 정보와 reload메서드)

  1. location.href : 참조주소나 url 반환
  2. location.hash : url의 #값을 반환
  3. location.hostname : url의 호스트 이름을 설정하거나 반환
  4. location.host : url의 호스트 이름과 포트 번호 반환
  5. location.protocol : url의 프로토콜 반환
  6. location.search : url의 쿼리값 반환
  7. location.reload() : 새로고침

history

사용자가 방문한 사이트의 기록을 남기고 이전 방문 사이트와 다음 방문 사이트로 다시 돌아갈 수 있는 속성 제공

  1. history.back() : 이전 방문 사이트로 이동
  2. history.forward() : 다음 방문 사이트로 이동
  3. history.go(이동숫자) : 이동 숫자에 -2를 입력하면 2단계 이전의 사이트로 이동
  4. history.length : 방문한 사이트의 기록을 남긴 총 수량

navigator

현재 방문자가 사용하는 브라우저 정보와 운영체제 정보를 제공하는 객체

  1. navigator.appCodeName : 현재 브라우저의 코드명 반환
  2. navigator.appName : 현재 브라우저의 이름 반환
  3. navigator.appVersion : 현재 브라우저의 버전 정보 반환
  4. navigator.language : 현재 브라우저가 사용하고 있는 언어 반환
  5. navigator.product : 현재 브라우저의 엔진 이름을 반환합니다.
  6. navigator.platform : 현재 컴퓨터의 운영체제 정보를 제공합니다.
  7. navigator.onLine : 온라인 상태 여부에 대한 정보를 제공합니다.
  8. navigator.userAgent : 브라우저와 운영체제의 종합 정보를 제공합니다.
